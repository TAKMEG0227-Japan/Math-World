# 🧮 Advanced Problem 1 

A certain job needs:

24 days if done by A alone
18 days if done by B alone
9 days if done by C alone
Starting on February 1st,
the work begins with a 1-person daily rotation:
A → B → C → A → B → ...

Question:
On what month and day will the job be completed?
